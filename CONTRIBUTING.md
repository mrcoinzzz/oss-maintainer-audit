# Contributing

Contributions are welcome, especially improvements that make the audit more useful for real maintainers.

## Good first contributions

- Add checks for another ecosystem
- Improve the wording of audit messages
- Add documentation examples
- Add tests for edge cases

## Development

Run the test suite:

```bash
python3 -m pytest
```

Run the CLI locally:

```bash
python3 -m oss_maintainer_audit.cli .
```

## Pull requests

Please keep pull requests focused and include tests when changing behavior.
